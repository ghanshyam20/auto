Bash Automation Scripts

The automation toolkit consists of several Bash scripts. Each script performs a specific task in the deployment pipeline.

Main Deployment Script



#!/bin/bash

echo "Starting WordPress automated deployment..."

chmod +x scripts/*.sh
chmod +x monitor/*.sh

./scripts/install_stack.sh
./scripts/setup_database.sh
./scripts/install_wordpress.sh
./scripts/install_wpcli.sh
./scripts/configure_wordpress.sh
./scripts/install_content.sh
./monitor/system_monitor.sh

Install Server Stack



sudo apt update -y
sudo apt install apache2 mariadb-server php php-mysql

Database Setup



CREATE DATABASE wordpress;
CREATE USER 'wpuser'@'localhost';

System Monitoring Script



echo "CPU usage"
uptime

echo "Memory usage"
free -h

echo "Disk usage"
df -h
