Project Setup Guide

System Requirements

  • Ubuntu or Debian Linux
  • Internet connection
  • sudo privileges
  • Git installed

Clone Repository

git clone https://github.com/ghanshyam20/automation_tool_kit.git

Enter Project Folder

cd automation_tool_kit

Run Deployment Script

sudo ./deploy.sh

Access Website

http://localhost
