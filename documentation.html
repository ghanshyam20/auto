Project Setup Guide

This guide explains how to run the automation toolkit from scratch.

Step 1 — Clone Repository


git clone https://github.com/ghanshyam20/automation_tool_kit.git

Step 2 — Enter Project Directory


cd automation_tool_kit

Step 3 — Run Deployment


sudo ./deploy.sh

Step 4 — Open Browser

Open the following URL:


http://localhost

After deployment you will see the automatically installed WordPress website.
