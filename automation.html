Automation Workflow

The automation process is controlled by a central deployment script. This script executes several modules responsible for different parts of the server setup.

  1. Clone project repository
  2. Run deployment script
  3. Install Apache, PHP and MariaDB
  4. Create WordPress database
  5. Install WordPress
  6. Configure WordPress using WP-CLI
  7. Add demo content
  8. Check system monitoring
